# خطة SEO + Google Analytics + Google Maps لعلامة سلة الحصاد

## الهدف
بناء حضور رقمي قوي ومتوافق مع هوية **سلة الحصاد** (الثقة منذ 1969، الطزاجة اليومية، خدمة قريبة مثل أهل البيت)، وتحويل الزيارات إلى طلبات مباشرة عبر واتساب/الهاتف.

## North Star KPI
- زيادة طلبات واتساب من الموقع بنسبة 30% خلال 90 يوم.

---

## خطة التنفيذ (Roadmap)

### المرحلة 1 (الأسبوع 1-2): الأساس التقني
- تفعيل GA4 وربط الأحداث الأساسية:
  - `view_item`, `add_to_cart`, `begin_checkout`, `contact_click`, `whatsapp_click`, `map_open`.
- ضبط SEO الأساسي:
  - Meta Title / Description لكل صفحة.
  - Canonical tags.
  - OpenGraph + Twitter cards.
  - Structured Data (LocalBusiness).
- تفعيل Google Search Console ورفع Sitemap.
- ضبط robots.txt ومراجعة noindex.

### المرحلة 2 (الأسبوع 3-4): SEO محلي Local SEO
- تحسين صفحة التواصل:
  - خريطة Google Maps واضحة.
  - عرض متوسط تقييم Google وأحدث الريفيوهات.
  - زر واضح "اكتب تقييمك".
- توحيد NAP عبر كل القنوات:
  - Name/Address/Phone متطابق حرفياً مع Google Business Profile.
- إنشاء صفحات Landing محلية:
  - "خضار وفواكه شفا بدران"، "توصيل خضار شفا بدران".

### المرحلة 3 (الأسبوع 5-8): المحتوى والبراند
- تقويم محتوى SEO أسبوعي (2 مقال/أسبوع):
  - أسعار موسمية، نصائح تخزين، وصفات سريعة.
- ربط المحتوى بالبراند:
  - قصص "من 1969"، صورة العائلة، معايير اختيار المنتجات.
- تحسين CTA في الموقع:
  - "اطلب عبر واتساب الآن" في كل الصفحات الرئيسية.

### المرحلة 4 (الأسبوع 9-12): التحسين المستمر
- Dashboard أسبوعي:
  - Sessions, CTR, Top Queries, WhatsApp Clicks, Conversion by page.
- A/B Testing:
  - نصوص الأزرار، عناوين الصفحات، بلوكات الثقة.
- خطة رفع تقييمات Google بشكل عضوي:
  - طلب تقييم بعد كل طلب مكتمل عبر رابط مباشر.

- [ ] ضغط الصور وتحسين Core Web Vitals.

### Analytics & Tracking
- [ ] تفعيل GA4 + Google Tag Manager.
- [ ] تعريف Conversion أساسي: الضغط على واتساب + إتمام الطلب.
- [ ] ربط GA4 مع Search Console وLooker Studio.

### Google Maps & Reviews
- [ ] ربط الموقع بـ Google Place ID الصحيح.
- [ ] عرض آخر المراجعات في صفحة التواصل (API + Cache).
- [ ] إنشاء رابط تقييم مباشر (Write Review Link).
- [ ] إرسال رسالة follow-up بعد التسليم تطلب التقييم.

### ملاحظة مهمة بخصوص "إرسال التقييمات الجديدة من الموقع إلى Google Maps"
لا يمكن **نشر تقييم Google Maps تلقائياً عبر API من موقعك** بالنيابة عن المستخدم. المسموح عملياً:
1) توجيه المستخدم لرابط "Write Review" الرسمي، أو
2) جمع تقييم داخلي داخل الموقع (اختياري) ثم مطالبة العميل بنشره بنفسه على Google.

## مؤشرات قياس النجاح (KPIs)
- CTR من نتائج البحث > 5% على الكلمات المحلية.
- نمو زيارات الصفحات المحلية 20% شهرياً.
- زيادة تقييمات Google الجديدة 15-25 تقييم/شهر.
- معدل تحويل النقر إلى واتساب من صفحات المنتجات > 8%.
